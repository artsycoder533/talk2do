Talk2Do

Talk2Do is a typing-free notes app that utilizes the Web Speech API. The Web Speech API is still in the experimental phase so compatibility is limited. Based on my testing, this app is not compatible with iPhones.


Directions: Allow the application to your microphone when the popup appears. Turn your sound on. Press the microphone button and say a command. Speak clearly and slowly.


Supported Commands:

  • "Show instructions"
  • "Close instructions"
  • "Add", Ex: "Add wash dishes"
  • "Edit", Ex: "Edit wash dishes"
  • "Delete", Ex: "Delete wash dishes"
  • "Delete all"
  • "Complete", Ex: "Complete wash dishes"
  • "Complete All"
  • "Undo", Ex: "Undo wash dishes"
  • "Undo all"
  • "Dark mode on"
  • "Dark mode off"
